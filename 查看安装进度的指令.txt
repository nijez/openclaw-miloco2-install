重要回复语言要求：
- 全程必须使用中文回复，包括中间进度、等待说明、异常解释和最终结果。
- 每次观察进度时只用中文简短说明当前状态。
- 每次中间汇报都必须带“检查时间”，不要只说继续等待。

请只执行下面这些只读检查：

```bash
LOG=/tmp/openclaw-miloco-install-current.log
STATE=/tmp/openclaw-miloco-install.state
PIDFILE=/tmp/openclaw-miloco-install.pid
CHECK_TIME="$(date '+%F %T %Z')"

echo "检查时间: $CHECK_TIME"
echo "状态文件: $STATE"
echo "日志文件: $LOG"

if [ -f "$PIDFILE" ]; then
  PID="$(cat "$PIDFILE" 2>/dev/null || true)"
  echo "后台任务编号: ${PID:-none}"
  if [ -n "$PID" ]; then
    ps -p "$PID" -o pid,etime,stat,cmd 2>/dev/null || true
  fi
else
  echo "后台任务编号: none"
fi

echo "--- 最近状态 ---"
tail -n 80 "$STATE" 2>/dev/null || echo "暂未找到状态文件"

echo "--- 最近日志 ---"
tail -n 120 "$LOG" 2>/dev/null || echo "暂未找到日志"

echo "--- 关键结果 ---"
grep -E 'SUCCESS|STEP_[0-9]+_DONE|ERROR|Failed|Traceback|timeout|timed out' "$STATE" "$LOG" 2>/dev/null | tail -n 80 || true
```

1. 本次检查时间。
2. 当前是否仍在安装。
3. 最后完成到哪一步。
4. 有没有看到 `SUCCESS_AFTER_RECONNECT`、`SUCCESS_ACTIVE` 或 `STEP_6_DONE`。
5. 最近有没有 ERROR / Failed / Traceback / timeout。
6. 我是否需要继续等待。
