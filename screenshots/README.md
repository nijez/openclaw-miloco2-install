# 教程截图拍摄清单

截图文件放在本目录。截图完成后，将教程页里的占位内容替换为对应图片。

| 文件名 | 截图内容 | 拍摄要点 |
|---|---|---|
| `01-server-create.png` | 腾讯云购买页，选中「小龙虾」应用模板 | 画面要能看到应用模板区域，并突出「小龙虾」模板已被选中。 |
| `02-server-login.png` | 腾讯云服务器列表或服务器详情页 | 突出「免密码登录」入口，方便用户知道从哪里打开终端。 |
| `03-mimo-key.png` | 腾讯云小龙虾应用设置里的 MiMo 订阅 Key 输入位置 | 不展示真实 Key；如页面已有输入内容，需要先打码。 |
| `04-terminal-progress.png` | 免密终端执行一键安装后的进度画面 | 截到安装进度正在滚动的阶段，能看到百分比和中文进度。 |
| `05-terminal-done.png` | 免密终端安装完成画面 | 截到「基础环境安装完成」和「回到小龙虾绑定米家账号」提示。 |
| `06-bind-link.png` | 小龙虾对话页给出米家授权链接 | 授权链接应是可点击样式；不要展示账号、Key 或其他敏感信息。 |
| `07-bind-code.png` | 米家网页授权完成后的授权码页面 | 授权码需要打码，只保留页面结构和复制位置。 |
| `08-home-select.png` | 小龙虾列出米家家庭供用户选择 | 确保能看到多个家庭列表，并突出选择馨光设备所在家庭。 |
| `09-skill-install.png` | 小龙虾正在安装馨光 Skill | 截到「正在安装馨光 Skill，请稍候」或等价进度提示。 |
| `10-skill-done.png` | 小龙虾提示馨光 Skill 已安装 | 截到「馨光 Skill 已安装，可以开始测试灯光」提示。 |
| `11-light-command.png` | 用户发送灯光测试语句，小龙虾回复观察提示 | 使用默认测试语句「客厅来个马尔代夫的海边日落」，并截到让用户观察灯光是否变化的回复。 |

拍摄前请确认所有服务器 IP、授权码、订阅 Key、账号信息都已打码。
