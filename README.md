# 云服务器 OpenClaw + Miloco 2.0 一键部署包

这是给腾讯云 / 阿里云 OpenClaw 应用模板使用的一键部署教程和脚本。

- 图文教程：https://nijez.github.io/openclaw-miloco2-install/
- 一键脚本：https://nijez.github.io/openclaw-miloco2-install/install-miloco-openclaw-cloud.sh
- 备用脚本：https://cdn.jsdelivr.net/gh/nijez/openclaw-miloco2-install@main/install-miloco-openclaw-cloud.sh

最简单用法：

1. 打开 OpenClaw 对话框。
2. 复制教程页面里的“一键部署指令”发给 OpenClaw。
3. 等待它完成 OpenClaw 更新、Miloco 2.0 部署和状态检查。

脚本默认不会保存 SSH 密码、token、MiMo API Key，也默认跳过微信、米家账号和 MiMo Key 绑定。
