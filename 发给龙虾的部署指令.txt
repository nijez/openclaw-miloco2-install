请你在这台服务器上执行「馨光 AI 设计灯光安装指导：第一阶段 OpenClaw + Miloco 2.0 底座部署」。默认环境是腾讯云轻量应用服务器的 OpenClaw 应用模板，OpenClaw 已预装。请不要自行搜索网络，不要改用源码安装、pip/pipx 安装、Docker 安装或手工解包安装。

重要回复语言要求：
- 全程必须使用中文回复，包括中间进度、等待说明、异常解释和最终结果。
- 不要用英文写 “I'll / Let me / Good / It's progressing” 这类过程说明。
- 命令输出或日志里的英文可以作为必要关键词保留，但你的解释必须翻译成中文。
- 当前前台回复只负责启动后台部署任务，不要长时间轮询，不要输出英文思考过程。

目标：
1. 下载固定脚本并校验版本。
2. 启动独立后台部署任务。
3. 后台任务负责 OpenClaw 最低版本检查、必要时升级、Miloco 2.0 部署和 Gateway 重启。
4. 默认跳过微信绑定、米家账号绑定、MiMo API Key 配置。
5. 如果 Gateway 重启导致 AgentChat 页面短暂断线，提示用户等待 1～3 分钟后重新连接。

当前阶段只部署底座，不安装馨光智能 Skill。馨光 Skill、设备白名单、淡彩光颜色规则和真机验收属于第二阶段。

请只执行下面这一份固定流程。不要在当前 AgentChat 前台会话里直接执行 systemctl restart、openclaw gateway restart、kill 或 pkill：

```bash
TARGET=/tmp/install-miloco-openclaw-cloud.sh
LOG=/tmp/openclaw-miloco-install-current.log
PIDFILE=/tmp/openclaw-miloco-install.pid
STATE=/tmp/openclaw-miloco-install.state
EXPECTED_VERSION=2026-06-25.2
rm -f "$TARGET"

for URL in \
  https://nijez.github.io/xingguang-ai-lighting-guide/install-miloco-openclaw-cloud.sh \
  https://raw.githubusercontent.com/nijez/xingguang-ai-lighting-guide/main/install-miloco-openclaw-cloud.sh \
  https://cdn.jsdelivr.net/gh/nijez/xingguang-ai-lighting-guide@main/install-miloco-openclaw-cloud.sh
do
  echo "尝试下载脚本: $URL"
  if curl -fsSL "$URL" -o "$TARGET" && grep -q "SCRIPT_VERSION=\"$EXPECTED_VERSION\"" "$TARGET"; then
    echo "脚本下载成功，版本 $EXPECTED_VERSION"
    break
  fi
  echo "当前源失败或版本不匹配，继续尝试下一个源"
done

grep -q "SCRIPT_VERSION=\"$EXPECTED_VERSION\"" "$TARGET" || { echo "固定脚本下载失败或版本不正确"; exit 1; }
chmod +x "$TARGET"

RUN_CONTEXT=agentchat_launcher \
DEPLOY_SUPERVISOR=1 \
INSTALL_ACTION=full \
RUN_SYSTEM_UPGRADE=0 \
OPENCLAW_UPDATE=auto \
INSTALL_EXTRA_PLUGINS=0 \
INSTALL_NONINTERACTIVE=1 \
LOG_FILE="$LOG" \
STATE_FILE="$STATE" \
PID_FILE="$PIDFILE" \
bash "$TARGET"
```
