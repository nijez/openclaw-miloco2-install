请你在这台服务器上部署 OpenClaw + Miloco 2.0。请严格按下面步骤执行，不要自行搜索网络、不要换安装路线、不要尝试源码安装、pip/pipx 安装、Docker 安装或手工解包安装。

目标：
1. 如果 OpenClaw 已预装，升级/配置 OpenClaw。
2. 如果 OpenClaw 未安装，自动安装 OpenClaw。
3. 安装并部署 Miloco 2.0。
4. 默认跳过微信绑定、米家账号绑定、MiMo API Key 配置。
5. 最后检查 OpenClaw gateway、Miloco 服务和插件状态。

唯一安装入口：
- 文件名：install-miloco-openclaw-cloud.sh
- 目标位置：/tmp/install-miloco-openclaw-cloud.sh
- 主下载地址：https://nijez.github.io/openclaw-miloco2-install/install-miloco-openclaw-cloud.sh
- 备用下载地址 1：https://cdn.jsdelivr.net/gh/nijez/openclaw-miloco2-install@main/install-miloco-openclaw-cloud.sh
- 备用下载地址 2：https://raw.githubusercontent.com/nijez/openclaw-miloco2-install/main/install-miloco-openclaw-cloud.sh

执行要求：
1. 下载固定脚本：

```bash
curl -fsSL https://nijez.github.io/openclaw-miloco2-install/install-miloco-openclaw-cloud.sh -o /tmp/install-miloco-openclaw-cloud.sh || curl -fsSL https://cdn.jsdelivr.net/gh/nijez/openclaw-miloco2-install@main/install-miloco-openclaw-cloud.sh -o /tmp/install-miloco-openclaw-cloud.sh || curl -fsSL https://raw.githubusercontent.com/nijez/openclaw-miloco2-install/main/install-miloco-openclaw-cloud.sh -o /tmp/install-miloco-openclaw-cloud.sh
chmod +x /tmp/install-miloco-openclaw-cloud.sh
```

2. 确认脚本是否存在、版本是否正确：

```bash
ls -l /tmp/install-miloco-openclaw-cloud.sh
grep '^SCRIPT_VERSION=' /tmp/install-miloco-openclaw-cloud.sh
```

3. 如果脚本不存在，或者版本低于 2026-06-23.9，请停止并告诉我：固定脚本下载失败。不要自己去网上找替代脚本。

4. 如果脚本存在且版本正确，执行：

```bash
INSTALL_ACTION=full INSTALL_NONINTERACTIVE=1 bash /tmp/install-miloco-openclaw-cloud.sh
```

5. 安装期间只需要观察输出。不要中途杀进程，不要并行启动第二个安装，不要切换到其他安装方法。除非脚本明确退出失败，否则继续等待。

6. 如果脚本失败，只允许做一次修复重试：

```bash
INSTALL_ACTION=repair RUN_SYSTEM_UPGRADE=0 INSTALL_NONINTERACTIVE=1 bash /tmp/install-miloco-openclaw-cloud.sh
```

7. 完成后执行验证：

```bash
miloco-cli service status
openclaw gateway status
openclaw plugins list | grep -i -C 2 miloco
ss -ltnp | grep -E ':(1810|18789)\b'
```

- 不要把 Miloco 面板直接暴露到公网。
