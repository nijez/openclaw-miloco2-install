常见问题

安装过程中页面异常怎么办？

通常是小龙虾后台服务正在重启。等待 1-3 分钟后刷新页面。

刷新后没有内容怎么办？

直接发送「查看安装进度」。

能不能重复发送一键部署指令？

不要重复发送。先用「查看安装进度」确认当前状态。

怎么查看安装进度？

复制本页「查看安装进度」内容，发给小龙虾。

安装完成后下一步做什么？

配置 MiMo API Key，绑定米家账号，确认馨光淡彩光设备已出现。

为什么还不能直接控制灯光？

当前页面验证的是第一阶段底座。馨光技能、设备白名单和真机控制验收属于后续阶段。

如果多次刷新后仍无法查看进度，请联系技术人员处理。
