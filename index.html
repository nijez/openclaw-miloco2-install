馨光 AI 设计灯光安装指导

准备好腾讯云轻量服务器的小龙虾（OpenClaw）应用模板和 MiMo API Key 后，把固定安装指令发给小龙虾对话页即可开始安装。

脚本版本：2026-06-25.3
2026-06-25.3 已完成真实腾讯云小龙虾应用模板服务器验证，可按本教程安装。

项目目标

完成安装后，用户可以在小龙虾里输入灯光需求，例如：“吊顶灯带，设计个马尔代夫灯光效果。”

小龙虾会通过馨光 Skill 控制馨光设备，生成对应的 AI 设计灯光效果。

安装流程

1

准备小龙虾服务器

购买腾讯云轻量应用服务器，并选择小龙虾应用模板。

2

配置小龙虾

开通小米 MiMo 大模型账号，获取并填写 MiMo API Key，绑定微信小龙虾。

3

安装馨光 AI 设计灯光

复制页面里的固定安装指令，发给小龙虾对话页。

4

配置并测试灯光效果

配置灯光能力、绑定米家账号、确认馨光设备、安装馨光 Skill，并输入灯光需求测试效果。

1. 准备小龙虾服务器

进入腾讯云小龙虾页面，购买腾讯云轻量应用服务器。创建服务器时选择小龙虾应用模板。服务器创建完成后，进入小龙虾对话页。

购买腾讯云小龙虾服务器

2. 配置小龙虾

进入小龙虾应用设置，填写 MiMo API Key。配置完成后，按页面提示绑定微信小龙虾。

绑定完成后，可以先给小龙虾发一句简单测试：“你好，帮我确认你已经可以正常工作。”

这里配置的是小龙虾自己的 MiMo API Key，用于让小龙虾能够正常对话和执行任务。
安装过程中不会要求填写你的 API Key。安装完成后，再按页面提示完成 MiMo API Key、米家账号和馨光设备配置。

3. 安装馨光 AI 设计灯光

复制页面里的固定安装指令，发给小龙虾对话页。小龙虾会在服务器后台安装馨光 AI 设计灯光所需能力。

如果安装过程中页面短暂异常，通常是小龙虾后台服务正在重启。请等待 1-3 分钟后刷新页面。刷新后如果是空白对话框，直接发送「查看安装进度」。
发给小龙虾的安装指令

4. 配置并测试灯光效果

安装完成后，还需要把 MiMo API Key 配置给馨光 AI 设计灯光能力。如果前面已经获取过 MiMo API Key，这里可以继续使用同一个 Key。

然后按提示绑定米家账号，并确认馨光设备已经出现在设备列表里。完成前面步骤后，即可进入馨光 Skill 安装与灯光测试。

1. 配置灯光能力 MiMo API Key

用于馨光 AI 设计灯光能力。

2. 绑定米家账号

用于读取并控制你的馨光设备。

3. 确认馨光设备

确认馨光设备已经出现在设备列表中。

4. 安装馨光 Skill

完成前面步骤后，即可进入馨光 Skill 安装与灯光测试。馨光 Skill 安装指令将继续补充。安装馨光 Skill 后，可以输入灯光需求测试效果。

两个 MiMo API Key 配置位置：

  • 小龙虾 MiMo API Key：用于小龙虾正常对话和执行任务。
  • 灯光能力 MiMo API Key：用于馨光 AI 设计灯光能力。
  • 这两个位置可以使用同一个 MiMo API Key。

  1. 小龙虾能理解灯光需求。
  2. 小龙虾能调用馨光 Skill。
  3. 馨光设备有实际变化。

常见问题

安装过程中页面短暂异常怎么办？

通常是小龙虾后台服务正在重启。请等待 1-3 分钟后刷新页面。刷新后如果是空白对话框，直接发送「查看安装进度」。

刷新后是空白对话框怎么办？

直接发送「查看安装进度」。

能不能重复发送一键部署指令？

不要重复发送。后台任务启动后，需要确认状态时只发送「查看安装进度」。

怎么查看安装进度？

复制本页「查看安装进度」内容，发给小龙虾。

安装完成后下一步做什么？

配置灯光能力所需的 MiMo API Key，绑定米家账号，确认馨光设备已经出现在设备列表中，然后安装馨光 Skill 并测试灯光效果。

为什么还不能直接控制灯光？

完成本页安装后，还需要继续完成 MiMo API Key、米家账号、馨光设备和馨光 Skill 配置，才能进行灯光控制测试。

MiMo API Key 要配置几次？

小龙虾本身需要配置一次。安装完成后，馨光 AI 设计灯光能力也需要配置一次。如果你使用的是同一个 Key，可以在两个位置填写同一个 Key。

米家账号什么时候绑定？

安装完成后，在配置灯光效果前按提示绑定米家账号。

馨光 Skill 什么时候安装？

完成前面步骤后，即可进入馨光 Skill 安装与灯光测试。馨光 Skill 安装指令将继续补充。

如果多次刷新后仍无法查看进度，请联系技术人员处理。
